# 🏥 MediBook - Online Appointment Booking System

## 📌 Project Overview
MediBook is a microservices-based healthcare appointment booking system where patients can search providers, view available slots, and book appointments seamlessly.

---

## 🏗️ Architecture (Planned)
- Microservices Architecture
- API Gateway (Spring Cloud Gateway)
- Authentication & Authorization (JWT + OAuth2)
- React Frontend (Upcoming)

---

# Use case 1 (Auth Service)

## 🔐 Auth Service Features
- User Signup (Register)
- User Login (JWT-based authentication)
- Password Encryption (BCrypt)
- Google OAuth2 Login Integration
- JWT Token Generation after OAuth Login

---

## 🧱 Tech Stack 
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- JWT (io.jsonwebtoken)
- OAuth2 Client (Google Login)
